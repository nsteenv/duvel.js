Solution

Les comparer

Valider les différences

Automatiser le processus

Comparons deux révisions

Quelles sont les différences?

On détecte

Les changements de styles, layout, couleurs (même minimes)

Les changements de contenu

Les exceptions JS / erreurs de chargement

On ne détecte pas

La cohérence des données

Les erreurs, mais les différences entre les deux versions

Comment ca marche

On déploie deux révisions de l'appli

On crawl avec duvel.js pour prendre les screenshots

On compare les images avec l'application pdiff

On valide les différences via une appli web

Et voilà!

Ressources

Duvel.js

Perceptual diff
